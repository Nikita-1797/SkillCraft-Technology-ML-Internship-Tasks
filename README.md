# SkillCraft-Technology-ML-Internship-Tasks

This repository contains **Machine Learning projects** completed as part of the **Skilcraft Technology Internship**.  
All projects are beginner-friendly Python implementations covering regression, clustering, image classification, and gesture recognition.

## Projects Included

## 1. House Price Dataset
- **Folder:** `House-Price-Dataset`
- **Description:** Predict house prices using **Linear Regression** based on features like square footage, bedrooms, and bathrooms.
- **Dataset:** `house_data.csv`
- **Script:** `house_price_Dataset.py`
- **Run:**
```bash
cd House-Price-Dataset
python house_price_Dataset.py


2. Mall Customer Dataset
Folder: Mall-Customer-Dataset
Description: Segment mall customers into clusters using K-Means clustering based on annual income and spending score.
Dataset: Mall_Customers.csv
Script: mall_customer_Dataset.py
Run:
cd Mall-Customer-Dataset
python mall_customer_Dataset.py

3. Dog-Cats Dataset
Folder: Dog-Cats-Dataset
Description: Classify images of cats and dogs using HOG features + SVM.
Dataset: dataset/cats and dataset/dogs
Script: Dog_Cats_Dataset.py
Run:
cd Dog-Cats-Dataset
python Dog_Cats_Dataset.py

4. Hand Gesture Recognition Dataset
Folder: Hand-Gesture-Recognition
Description: Recognize hand gestures using image processing and machine learning techniques.
Dataset: dataset/gesture1, gesture2, ... (image folders)
Script: hand_gesture.py
Run:
cd Hand-Gesture-Recognition
python hand_gesture.py
